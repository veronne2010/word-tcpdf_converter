# Word DOCX → TCPDF Converter

Tool web (GitHub Pages) che converte file DOCX in template PHP per TCPDF.

## 🚀 Funzionalità

- Upload DOCX
- Conversione HTML lato browser
- Generazione template PHP TCPDF
- Supporto placeholder {{campo}}
- Header/Footer base
- Numerazione pagina automatica

## 🧪 Uso

1. Apri la webapp
2. Carica file .docx
3. Clicca Converti
4. Scarica file PHP
5. Usalo nel tuo progetto TCPDF

## 🧱 Stack

- HTML
- JavaScript
- Mammoth.js
- FileSaver.js

## ⚠️ Nota

Non richiede backend. Funziona interamente su GitHub Pages.
